# VideoPoker

# Welcome to Video Poker!

# The player starts with $100.
# Each bet is $1.
# Payout is as follows:
# Royal Flush             250
# Straight Flush          50
# Four of a Kind          25
# Full House              6
# Flush                   5
# Straight                4
# Three of a Kind         3
# Two Pair                2
# Pair of Jacks or better 1
  
# The game will go on until the player runs out of money.
